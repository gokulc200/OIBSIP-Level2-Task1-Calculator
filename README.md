# OIBSIP-Level2-Task1-Calculator

<h1>Overview 📖</h1>
<p>The calculator application within this repository encompasses a wide range of mathematical functionalities. It includes all basic arithmetic operations such as addition, subtraction, multiplication, and division, making it suitable for simple calculations. However, it goes beyond the basics and provides advanced capabilities such as exponentiation, square roots, logarithms, and trigonometric functions, catering to more complex mathematical requirements.</p>

<h1>Technology Used 👨🏻‍💻</h1>
<p>I have built this project using the following tools & techniques:</p>
🔴 HTML<br/>
🔴 CSS<br/>
🔴 JavaScript<br/>
🔵 Flexbox<br/>
🔵 Font Awesome Icons<br/>
🟡VSCode Editor

<h1>Features💡</h1>
<h3>🟣Basic and Advanced Operations:</h3>
<p>The calculator supports all fundamental arithmetic operations such as addition, subtraction, multiplication, and division. Additionally, it offers advanced functionalities like exponentiation, square roots, logarithms, trigonometric functions, and more.</p>

<h3>🟣User-Friendly Interface:</h3> The calculator boasts an intuitive and user-friendly interface that enables smooth interaction. It provides a responsive design that adapts seamlessly to different screen sizes and devices.

<h3>🟣Error Handling:</h3> The calculator implements comprehensive error handling to ensure accurate calculations. It detects and gracefully handles invalid inputs, division by zero scenarios, and other potential errors, providing informative feedback to users.

<h3>🟣Light and Dark Mode</h3> It includes a highly sought-after feature: a light and dark mode option for the calculator application. This feature provides users with the flexibility to choose between two distinct visual themes, enhancing their user experience and personalization.
  
<h3>🟣Customization Options:</h3> This repository offers customization options to tailor the calculator's appearance and behavior according to specific requirements. Developers can easily modify the color schemes, button layouts, and functionality to match their application's design and needs.

<h3>🟣Cross-Platform Compatibility:</h3>The calculator is designed to work across multiple platforms and operating systems, including Windows, macOS, Linux, iOS, and Android. It leverages web technologies or platform-specific frameworks to ensure maximum accessibility.

<h3>🟣Extensible and Well-Structured Codebase:</h3> The repository follows best practices in software development, employing modular and well-documented code. This promotes code reuse, maintainability, and enables developers to extend the calculator's functionalities with ease.
